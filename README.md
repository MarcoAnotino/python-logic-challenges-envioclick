# Python Backend Logic Challenge

Caso Práctico Backend — Prueba técnica de habilidades y lógica en **Python 3.13**.

Este repositorio contiene ejercicios resueltos de forma independiente,
cada uno con su propia forma de ejecución y documentación.

## Reglas cumplidas

- Python **3.13**
- Sin librerías externas (solo librería estándar)
- Sin funciones “facilitadoras” prohibidas para búsqueda/manipulación
  (por ejemplo: `in`, `find`, `index`, `regex`, `sort`, etc.)
- Código alineado a estándares **PEP8**
- Uso de tipado y documentación con docstrings
